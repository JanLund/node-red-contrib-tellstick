Send data to Telldus Tellstick.

There are 3 ways of passing data to an out node:

  • Configure the device, method (turn on/turn off/dim) and dimlevel (0-255) if a dim capable device on the output node itself.
  • Send the data in the msg sent to the out node. msg object sent into this node may contain:
    • device, integer, device ID.
    • method, string, one of: turnoff, turnon, dim, bell, learn
    • dimlevel: integer 0-255, if method is set to dim. 255 is 100% and 0 is off.
  • A combination of above. E.g set the device to Red lightsaber in the configuration dialog but send method: turnon or method: turnoff in the msg to the out node. Whatever is sent in the msg will always override the settings set in the node.

See the online documentation for more help.
